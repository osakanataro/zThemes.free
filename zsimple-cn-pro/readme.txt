== Installation ==

= Via WordPress Admin =
1. From your sites admin, go to Themes > Install Themes
2. In the search box, type 'zSimple' and press enter
3. Locate the entry for 'zSimple' (there should be only one) and click the 'Install' link
4. When installation is finished, click the 'Activate' link

= Manual Install =
1. Unzip this file into your mywebsite.com/wp-content/themes/ directory, should look like mywebsite.com/wp-content/themes/zsimple/
2. Go into your Wordpress Admin, navigate to 'Appearance > Themes'
3. Find the zSimple listing on this page and click 'Activate'

== Theme Options ==

zSimple has theme options page: Go into your Wordpress Admin, navigate to 'Appearance > Customize'
